So you think you know Specificity?

Did you know that you can double up a class name to increase specificity of a selector?

I'm an example! √

<div class="blue red">I'm an example! √</div>

Did you know that in some browsers you can surpass any specificity?

I'm an example! √

<div class="surpass" style="background:black!important">
  I'm an example! √
</div>

So you think you know positioning?

Did you know that you can center anything with position and transforms?

I'm an example! √ 

<div class="srslyCenter">
  <span>I'm an example! √</span>
</div>
